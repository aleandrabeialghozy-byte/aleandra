# Biodata Portfolio Website

Website portfolio pribadi dengan desain modern dan elegan.

## Fitur
- Responsive design
- Smooth scrolling
- Parallax effects
- Gallery foto
- Timeline pendidikan
- Animasi interaktif

## Teknologi
- HTML5
- CSS3
- Vanilla JavaScript
- Modern CSS (Grid, Flexbox)

## Setup
1. Clone repository
2. Ganti foto di folder assets/images/
3. Edit konten di index.html
4. Deploy ke hosting

## Browser Support
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+# aleandra
